# 187GSAP

Build GSAP timelines, ScrollTrigger pinning/scrubbing, and performance-safe animation patterns for React and Next.js.

## When to use

- GSAP timeline choreography.
- ScrollTrigger pinning, scrub, or progress-linked effects.
- Easing, stagger, and reduced-motion fallback planning.

## Output

1. Timeline spec (targets, duration, easing, stagger).
2. ScrollTrigger map (trigger, start/end, scrub, pin).
3. Easing and performance notes.
4. Reduced-motion fallback.
5. Next actions.

## Routes

- `187MOTION` for reusable hooks.
- `187SCROLL` when scroll drives a 3D camera/scene.
- `187ACCESS+` for vestibular or photosensitive concerns.
