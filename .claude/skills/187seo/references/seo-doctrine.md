# 187SEO Doctrine

## Philosophy

SEO exists to connect real user intent with genuinely useful, technically sound content. Every recommendation must serve the searcher first and the search engine second.

## Priority framework

1. **User intent first.** Every page must match the intent behind the queries it targets. If intent is unclear, research before optimizing.
2. **Technical foundation.** Crawlability, indexability, security, speed, and mobile usability are prerequisites for everything else.
3. **E-E-A-T.** Demonstrate Experience, Expertise, Authoritativeness, and Trustworthiness through original insight, credible authorship, and transparent sourcing.
4. **Accessibility as SEO input.** Semantic HTML, alt text, readable contrast, keyboard navigation, and screen-reader compatibility strengthen both accessibility and search signals.
5. **Sustainable long-term practices.** Build durable organic value. Avoid shortcuts that trade long-term trust for short-term ranking gains.

## Decision rule

When in doubt, choose the action that improves user trust and content utility over the action that merely chases an algorithmic signal.
