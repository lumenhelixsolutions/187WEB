# 187FREE Skill Ideas

Twenty skill concepts for 187FREE. Each is a callable micro-skill.

## 1. Free Launch Architect

Generate a complete free launch stack for landing pages, docs, static demos, portfolios, and public project pages.

Checks: GitHub Pages, Cloudflare Pages, Netlify, Vercel, Firebase Hosting, Render static, docs hosting.

## 2. Free Fullstack MVP Builder

Generate a no-cost MVP architecture using hosted frontend, serverless/backend, auth, database, storage, and monitoring.

Checks: Vercel/Netlify/Cloudflare Workers, Supabase, Firebase, Neon, Turso, Cloudflare D1/R2, Render, Fly.io where applicable.

## 3. Free Domain/Subdomain Scout

Find safe free domain-like options and subdomains.

Checks: GitHub Pages subdomain, Vercel/Netlify/Cloudflare Pages subdomains, EU.org-style reputable options if still valid.

Warn: Avoid sketchy free TLDs for serious projects.

## 4. Free API Miner

Find free/public APIs for a project and classify reliability, auth, quota, CORS, license, and allowed use.

Checks: free-for.dev API sections, public API directories, official API docs.

## 5. Free Database Selector

Choose free DB/storage options by workload.

Modes: SQL, Postgres, SQLite edge, document DB, KV, vector, blob/object, file storage.

Checks: Supabase, Neon, Turso, MongoDB Atlas, Firebase, Cloudflare D1/KV/R2, Upstash.

## 6. Free Auth and Identity Scout

Find free auth options and pick by risk.

Checks: Supabase Auth, Firebase Auth, Clerk, Auth0, Cloudflare Access, OAuth provider constraints.

Sensitive-data rule: Prefer self-hosted/local or privacy-first when needed.

## 7. Free Monitoring and Observability Scout

Find free uptime, error tracking, logs, analytics, status pages, and performance monitoring.

Checks: UptimeRobot-style services, Sentry, Grafana Cloud, Better Stack, Plausible/community alternatives, browser perf tooling.

## 8. Free AI and Local Model Scout

Find free AI APIs, open models, local model tools, embeddings, OCR, STT/TTS, image utilities, and coding assistants.

Rules: Check privacy, usage limits, prompt/data retention, commercial limits, local fallback.

## 9. Free Accessibility and Assistive Tech Scout

Find open-source/free accessibility tools, AAC tools, readability tools, captioning, TTS/STT, browser extensions, cognitive-accessibility supports.

Rules: Consent-first, privacy-first, no surveillance, no hidden caregiver mode.

## 10. Free Security and Compliance Scout

Find free scanners and security hygiene tools.

Checks: GitHub Dependabot, CodeQL, secret scanning, OWASP ZAP, Snyk free, Semgrep, Trivy, SSL checkers, Mozilla Observatory.

## 11. Free Forms and Intake Builder

Build no-cost forms, surveys, research intake, civic reports, client intake, and feedback workflows.

Checks: Google Forms, Tally, Airtable/Notion-like free tiers, static form handlers, self-hosted alternatives.

Privacy rule: Do not use for sensitive data unless reviewed.

## 12. Free Docs and Knowledge Base Builder

Build free documentation, guidebooks, help centers, API docs, research pages, and Obsidian publish alternatives.

Checks: MkDocs, Docusaurus, Astro Starlight, GitHub Pages, Read the Docs, VitePress.

## 13. Free Data Visualization Lab Scout

Find free charting, dashboard, map, graph, and data-exploration tools.

Checks: Observable, D3, Vega-Lite, Plotly, Apache ECharts, Leaflet, MapLibre, Kepler.gl.

## 14. Free Media Utility Scout

Find free image/audio/video utilities for compression, conversion, captions, thumbnails, waveform, spectrogram, and social assets.

Checks: FFmpeg, ImageMagick, SVGOMG, browser APIs, open-source web tools.

Copyright rule: Respect licenses and do not scrape paid assets.

## 15. Free Dataset Scout

Find public/open datasets for research, civic, science, education, accessibility, AI, and product prototyping.

Checks: Data.gov, World Bank, NASA, NIH/NCBI, Kaggle public, Zenodo, OpenAlex, Wikidata, OpenStreetMap.

## 16. Free Code Intelligence Scout

Find packages, repos, examples, issues, prior art, templates, and comparable open-source implementations.

Checks: GitHub API, npm Registry, PyPI JSON API, Libraries.io, crates.io, package metadata, license, maintainer activity.

## 17. Free Automation and Workflow Scout

Find free automation options for CI/CD, scheduled jobs, webhooks, notifications, and low-code automations.

Checks: GitHub Actions, cron-job services, Cloudflare Workers cron triggers, n8n self-host, Node-RED, Make/Zapier free limits if applicable.

## 18. Free Classroom/Lab Builder

Find no-cost tools for educational labs, browser notebooks, interactive demos, assignments, quizzes, and reproducible experiments.

Checks: JupyterLite, Pyodide, Observable, GitHub Classroom, Binder alternatives, static lab pages.

## 19. Free Civic Tech Stack Scout

Find free/low-cost stack for civic evidence, public records, timelines, forms, dashboards, and public transparency pages.

Rules: Privacy-first, records-preservation first, no scraping restricted systems.

## 20. Free Client Budget Optimizer

Given a client project, recommend the cheapest viable stack and identify where paid upgrades will likely become necessary.

Output: free now, paid later, never use free for this component.
