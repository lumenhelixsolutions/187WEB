# WCAG+ Pre-Publish Checklist

Use this checklist before publishing any public page, form, course, quiz, or docs page.

## Perceivable

- [ ] Meaningful images have descriptive alt text; decorative images use null alt.
- [ ] Video has captions and transcript.
- [ ] Audio has transcript.
- [ ] Color is not the only means of conveying information.
- [ ] Text and images of text meet contrast ratios (4.5:1 normal, 3:1 large/UI components).
- [ ] Content remains usable when zoomed to 200%.

## Operable

- [ ] All functionality works with keyboard only.
- [ ] Focus order is logical and visible.
- [ ] Skip links or landmarks are available for navigation.
- [ ] Users can pause, stop, or hide moving/blinking content.
- [ ] No seizure-risk flashing.
- [ ] Touch targets are at least 44 by 44 CSS pixels where feasible.

## Understandable

- [ ] Page language is declared.
- [ ] Form errors are identified and described.
- [ ] Error prevention is offered for destructive or legal/financial actions.
- [ ] Navigation is consistent across pages.
- [ ] Jargon is defined or avoided.

## Robust

- [ ] Markup validates without critical errors.
- [ ] Custom controls expose name, role, and value.
- [ ] Status messages are announced without moving focus.

## Plus layers

- [ ] Cognitive load is chunked and predictable.
- [ ] Neurodivergent users have clear expectations and reduced ambiguity.
- [ ] Motion respects `prefers-reduced-motion`.
- [ ] Sensory density can be reduced.
- [ ] Plain-language summary is available for complex content.
