# 187TYPE

Design reactive, variable, and three-dimensional typography with readability and accessibility guardrails.

## When to use

- Kinetic headlines and hero type.
- Variable-font axis animation.
- 3D text scenes with R3F/Three.js.

## Output

1. Type motion spec.
2. Variable-font axis plan.
3. 3D text render notes.
4. Readability / accessibility review.
5. Next actions.

## Routes

- `187GSAP` for timeline/scroll sequencing.
- `187HERO` for full-screen hero integration.
- `187ACCESS+` for motion/readability review.
