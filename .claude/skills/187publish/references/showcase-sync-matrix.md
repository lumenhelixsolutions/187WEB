## GitHub Pages showcase

The static showcase must include:

- Hero with expanded suite tagline.
- Skill grid linking to each app route or doc.
- Recent changelog entry.
- Install command snippet.
- Accessibility and inclusion statements.
- Link to GitHub repository.

## Sync verification

- [ ] Every first-class skill has a showcase tile.
- [ ] Every tile links to a live app route or doc.
- [ ] Hero copy matches the canonical homepage copy.
- [ ] Changelog snippet is current.
- [ ] No broken links in the showcase.
