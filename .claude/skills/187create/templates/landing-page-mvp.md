# Landing Page MVP

Create a one-section landing page for **[offer]** targeting **[audience]**.

## Required elements

- Headline: clear, benefit-driven, audience-specific.
- Subhead: one sentence that expands the promise or handles the main objection.
- Social proof placeholder: testimonial, logo bar, or trust marker.
- CTA: single primary action with a clear verb.
- Visual direction: hero image, illustration, or background treatment.

## Accessibility checks

- [ ] Color contrast meets WCAG 2.1 AA for all text.
- [ ] Alt text provided for every image.
- [ ] Focus order lands on the CTA logically.
- [ ] Reduced-motion fallback for any motion.
- [ ] No autoplay video or audio.
