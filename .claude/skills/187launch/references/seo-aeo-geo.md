# SEO, AEO & GEO

## Traditional SEO

- Target 1–2 primary keywords per page.
- Write a compelling `<title>` (≤60 chars) and meta description (≤160 chars).
- Use semantic HTML and internal links.
- Build backlinks from directories, guest posts, and partnerships.

## AI Search Optimization (AEO)

- Answer specific questions directly in H2/H3 sections.
- Use structured data (JSON-LD FAQ, HowTo).
- Optimize for featured snippets: concise definition + bullet list.

## Generative Engine Optimization (GEO)

- Earn mentions in LLM training data via high-authority citations.
- Publish original research, datasets, and definitions.
- Build brand presence so models associate your product with the category.

## Quick Checklist

- [ ] Keyword mapped to every core page.
- [ ] JSON-LD structured data added.
- [ ] Internal links between related content.
- [ ] Page speed < 3s on mobile.
- [ ] Backlink outreach plan in place.
