# idea-spark — SPARK

**Module:** **SPARK** · Parent: [`187web-ecosystem`](../187web-ecosystem/SKILL.md)

Phase 1 foundation stub. Protocol: `frame → diverge → critic → score → combine → converge → ADR`.

## Manual triggers

`/187 spark`, `SPARK`, `brainstorm`, idea-spark

## When to use

Unresolved architecture or product options needing 3–5 distinct structures.

## When not to use

Settled branding, logo, mascot geometry (forbidden to brainstorm rebrand).

## Output contract

1. Options (3–5)  
2. Criteria scores  
3. Selected option + rationale summary  
4. Rejected reasons  
5. `decision_record` YAML fields  

## Safety

No brand reinterpretation. No illegal or unsafe “creative” paths.

## Acceptance tests

1. “Spark options for hub election” → ≥3 options + convergence.  
2. “Spark new logo colors” → refuse / redirect to brand lock.
