# 187FREE Gotcha Ledger

Check every item that applies before finalizing a recommendation.

## Account & Billing

- [ ] Credit card required?
- [ ] Free tier expiration after trial?
- [ ] Account verification required (phone, ID, company)?
- [ ] Geographic / region restrictions?
- [ ] Only free for open-source or nonprofits?

## Compute & Sleep

- [ ] Idle sleep or shutdown after inactivity?
- [ ] Cold-start latency?
- [ ] CPU / RAM / execution time limits?
- [ ] Build minutes / concurrency limits?
- [ ] Concurrent user / connection limits?

## Data & Storage

- [ ] Storage capacity limit?
- [ ] Database row / document / compute limit?
- [ ] Data retention policy?
- [ ] Backup / restore available?
- [ ] Export path? (JSON, SQL dump, etc.)
- [ ] Egress / bandwidth limits?

## API & Usage

- [ ] API request quota?
- [ ] Rate limits?
- [ ] Commercial-use restrictions?
- [ ] Attribution required?
- [ ] CORS / embedding restrictions?

## Legal & Lock-in

- [ ] Vendor lock-in? (proprietary formats, no export)
- [ ] License compatibility?
- [ ] Patent / export control concerns?
- [ ] Terms-of-service changes risk?

## Privacy & Telemetry

- [ ] Telemetry enabled by default?
- [ ] Data shared with third parties?
- [ ] Analytics / tracking injected?
- [ ] No HIPAA/GDPR/SOC-2 guarantees on free tier?

## Accessibility

- [ ] UI accessible to keyboard / screen reader?
- [ ] Color contrast adequate?
- [ ] Cognitive load reasonable?
- [ ] Documentation clear for non-experts?

## Reliability

- [ ] Project / company health? (funding, maintenance)
- [ ] Recent commits / releases?
- [ ] Public status page?
- [ ] Community support only?
