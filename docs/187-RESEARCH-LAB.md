# 187WEB Research Lab Stack

187WEB includes a research and laboratory layer for scientific standards, public data APIs, notebooks, GAP/Colab workflows, reproducibility, and release packets.

## Stack

```text
187RESEARCH  source maps
187SCI       claim discipline
187LABS      protocols and experiments
187DATA      datasets and public DB workflows
187API       public API contracts
187BENCH     benchmarks and reproducibility
187NB        notebook protocol
187COLAB     Google Colab execution profile
187GAP       computational algebra profile
187META      metadata and citation
187PROV      provenance and run lineage
187CRATE     RO-Crate / release packet
RRP          Research Release Packet
```
