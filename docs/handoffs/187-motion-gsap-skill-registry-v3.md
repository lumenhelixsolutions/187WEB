## 🧠 CORE ARCHITECTURE (APPLIED TO ALL /187 SKILLS)

Every command in this registry inherits the following production-safe guardrails:

| Guardrail | Implementation |
| :--- | :--- |
| **Auto-Cleanup** | All animations wrap in `gsap.context()` to prevent React/Vue memory leaks. On unmount, `context.revert()` is called. |
| **Motion-Safe** | Detects `window.matchMedia('(prefers-reduced-motion)')`. If active, all animations instantly resolve to their final state (no tweens). |
| **Performance** | Applies `will-change: transform, opacity` only during animation and removes it via `onComplete`. |
| **Responsive** | `ResizeObserver` or `matchMedia` rebinds breakpoints; debounced by 250ms. |
| **Dual-Trigger** | Every skill responds to: (1) Explicit slash command (e.g., `/187confetti`), and (2) Inline natural language (e.g., *"burst confetti from cursor"*). |
| **Hybrid Ready** | When Three.js/WebGL is detected, the skill auto-upgrades to GPU-accelerated rendering (see *Hybrid Expansion* section). |
