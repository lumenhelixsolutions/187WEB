# 187VIZ

Build Three.js data visualizations for networks, graphs, force layouts, and spatial data stories.

## When to use

- 3D network or node-edge graph.
- Animated data updates and selection states.
- Spatial data story with camera guidance.

## Output

1. Data schema and source plan.
2. Layout algorithm choice.
3. Interaction plan.
4. Render performance notes.
5. Next actions.

## Routes

- `187MODEL` for detailed 3D node representations.
- `187SCROLL` for scroll-narrative integration.
- `187ACCESS+` for screen-reader / keyboard graph navigation.
