# 187AUDIO

Build Web Audio-driven Three.js geometry that reacts to frequency data, beats, and microphone input.

## When to use

- Audio-reactive mesh or particle systems.
- Frequency-band synced visuals.
- Microphone-driven motion with permission handling.

## Output

1. Audio node graph.
2. Frequency-band mapping.
3. Geometry reactivity spec.
4. Permission and performance notes.
5. Next actions.

## Routes

- `187VIZ` for data-driven visualization output.
- `187MOTION` for reusable audio-reactive hooks.
- `187ACCESS+` for auto-play audio or flashing motion.
