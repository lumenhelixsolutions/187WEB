# 187FREE Launch Checklist — {{PROJECT_NAME}}

## Pre-launch

- [ ] Need defined and documented
- [ ] Best free pick selected and justified
- [ ] Two backup options identified
- [ ] Complete free stack recipe written
- [ ] Gotcha checks completed
- [ ] Privacy / sensitivity review completed
- [ ] Upgrade path documented
- [ ] Custom domain or subdomain configured
- [ ] SSL / HTTPS enabled
- [ ] Environment variables and secrets secured
- [ ] No API keys committed to repo

## Quality

- [ ] Smoke test on free tier
- [ ] Mobile / responsive check
- [ ] Keyboard / screen-reader accessibility check
- [ ] Performance budget reviewed
- [ ] Error tracking configured
- [ ] Uptime monitoring configured
- [ ] Analytics privacy settings reviewed

## Legal / Compliance

- [ ] License compatibility verified
- [ ] Terms of service reviewed
- [ ] Attribution requirements met
- [ ] Privacy policy / consent flow in place if collecting data
- [ ] No scraping of restricted systems

## Post-launch

- [ ] Status page or health endpoint live
- [ ] Backup / export job tested
- [ ] Team knows free-tier limits
- [ ] Upgrade trigger conditions documented
- [ ] Launch notes captured for future reference
